# 财务指标解读指南

## 一、盈利能力指标

### 1.1 毛利率 (Gross Profit Margin)

```
公式: (营业收入 - 营业成本) / 营业收入 × 100%

数据来源: get_financial_indicator → grossprofit_margin
```

**解读框架**:

| 毛利率区间 | 行业特征 | 护城河含义 |
|-----------|---------|-----------|
| >60% | 奢侈品、软件、医药 | 强品牌/专利壁垒 |
| 40-60% | 消费品、化工 | 可能有一定壁垒 |
| 20-40% | 制造业、零售 | 壁垒较弱 |
| <20% | 大宗商品、贸易 | 无明显壁垒 |

**趋势分析**:
- 上升: 产品升级、定价权增强、成本优化
- 稳定: 竞争格局稳定
- 下降: 价格战、成本上涨、竞争加剧

### 1.2 净利率 (Net Profit Margin)

```
公式: 净利润 / 营业收入 × 100%

数据来源: get_financial_indicator → netprofit_margin
```

**解读框架**:

| 净利率区间 | 评价 | 常见行业 |
|-----------|------|---------|
| >20% | 极佳 | 白酒、软件、医药 |
| 10-20% | 良好 | 消费品、金融 |
| 5-10% | 一般 | 制造业 |
| <5% | 较弱 | 零售、贸易 |

### 1.3 ROE (净资产收益率)

```
公式: 净利润 / 净资产 × 100%

数据来源: get_financial_indicator → roe

杜邦分解:
ROE = 净利率 × 资产周转率 × 权益乘数
```

**ROE与护城河关系**:

| ROE持续水平 | 护城河判断 |
|------------|-----------|
| >25% | 宽护城河，极强竞争优势 |
| 20-25% | 宽护城河 |
| 15-20% | 窄护城河 |
| 10-15% | 无明显护城河 |
| <10% | 竞争激烈，无护城河 |

**ROE驱动因素判断**:

| 主要驱动 | 护城河来源 | 可持续性 |
|---------|-----------|---------|
| 高净利率 | 品牌、定价权 | 高 |
| 高周转率 | 成本优势、效率 | 中 |
| 高杠杆 | 金融杠杆 | 低 (需谨慎) |

### 1.4 ROA (总资产报酬率)

```
公式: 净利润 / 总资产 × 100%

数据来源: get_financial_indicator → roa
```

**解读**:
- ROA > 10%: 资产利用效率极高
- ROA 5-10%: 良好
- ROA < 5%: 一般

## 二、运营效率指标

### 2.1 资产周转率

```
公式: 营业收入 / 总资产

解读:
- 周转率高 → 资产轻、效率高
- 周转率低 → 资产重、需关注资本回报
```

### 2.2 存货周转天数

```
公式: 365 / (营业成本 / 平均存货)

解读:
- 天数下降 → 运营效率提升
- 天数上升 → 可能库存积压
```

### 2.3 应收账款周转天数

```
公式: 365 / (营业收入 / 平均应收账款)

解读:
- 天数下降 → 议价能力强
- 天数上升 → 下游客户议价能力强
```

## 三、偿债能力指标

### 3.1 资产负债率

```
公式: 总负债 / 总资产 × 100%

数据来源: get_financial_indicator → debt_to_assets
```

**行业差异**:

| 行业类型 | 合理范围 | 说明 |
|---------|---------|-----|
| 银行/金融 | 90%+ | 行业特性 |
| 房地产 | 60-80% | 资本密集 |
| 制造业 | 40-60% | 适中 |
| 科技/消费 | <40% | 轻资产 |

### 3.2 流动比率

```
公式: 流动资产 / 流动负债

数据来源: get_financial_indicator → current_ratio

解读:
- >2: 流动性充足
- 1.5-2: 健康
- 1-1.5: 需关注
- <1: 流动性风险
```

### 3.3 速动比率

```
公式: (流动资产 - 存货) / 流动负债

数据来源: get_financial_indicator → quick_ratio

解读:
- >1: 短期偿债能力强
- 0.5-1: 需关注存货变现能力
- <0.5: 短期流动性风险
```

## 四、现金流指标

### 4.1 经营现金流与净利润比

```
公式: 经营活动现金流净额 / 净利润

数据来源:
- get_cashflow_statement → n_cashflow_act
- get_income_statement → n_income
```

**解读框架**:

| 比率 | 含义 | 护城河影响 |
|-----|------|----------|
| >1.2 | 现金流质量极佳 | 正面 |
| 1.0-1.2 | 现金流健康 | 正面 |
| 0.8-1.0 | 可接受 | 中性 |
| 0.5-0.8 | 需关注 | 负面 |
| <0.5 | 现金流质量差 | 强烈负面 |

### 4.2 自由现金流

```
公式: 经营现金流 - 资本支出

资本支出 = 购建固定资产、无形资产和其他长期资产支付的现金

解读:
- 持续正值 → 财务自由度高
- 持续负值 → 需要外部融资
```

### 4.3 资本开支强度

```
公式: 资本支出 / 营业收入 × 100%

解读:
- 下降趋势 → 行业供给收缩，可能改善
- 上升趋势 → 产能扩张，竞争可能加剧
```

## 五、成长性指标

### 5.1 营收复合增长率 (Revenue CAGR)

```
公式: (期末收入/期初收入)^(1/年数) - 1

数据来源: get_financial_metrics → metrics: ["revenue"], calc_type: "cagr"
```

**解读**:
- >25%: 高速成长
- 15-25%: 快速成长
- 5-15%: 稳定成长
- <5%: 成熟期

### 5.2 净利润复合增长率 (Profit CAGR)

```
公式: (期末净利润/期初净利润)^(1/年数) - 1

数据来源: get_financial_metrics → metrics: ["profit"], calc_type: "cagr"
```

**与营收增速对比**:
- 利润增速 > 营收增速 → 盈利能力提升
- 利润增速 < 营收增速 → 盈利能力下降
- 利润增速 = 营收增速 → 盈利能力稳定

## 六、估值参考指标

### 6.1 市盈率 (PE)

```
公式: 股价 / 每股收益

解读:
- 低PE + 高ROE + 低增速 = 价值股
- 高PE + 高增速 = 成长股
- 高PE + 低增速 = 高估
```

### 6.2 市净率 (PB)

```
公式: 股价 / 每股净资产

特别适用于:
- 银行、保险 (资产驱动)
- 周期股 (资产价值)
- 重资产行业
```

### 6.3 PEG

```
公式: PE / 净利润增长率

解读:
- PEG < 1: 可能低估
- PEG = 1: 合理
- PEG > 1: 可能高估
```

## 八、财务数据质量检查

### 检查清单

```markdown
□ 收入与现金流是否匹配？
  - 经营现金流 / 净利润 > 0.8

□ 应收账款是否异常增长？
  - 应收账款增速 < 营收增速

□ 存货是否积压？
  - 存货周转天数是否上升

□ 商誉是否过高？
  - 商誉 / 净资产 < 30%

□ 关联交易是否合理？
  - 关联交易占比是否过高

□ 财务数据是否连续？
  - 是否有异常波动
```

### 红旗信号

| 信号 | 可能问题 |
|-----|---------|
| 现金流与利润长期背离 | 盈利质量差 |
| 应收账款激增 | 可能提前确认收入 |
| 存货异常上升 | 产品滞销 |
| 毛利率突然提升 | 可能会计操纵 |
| 经营现金流持续为负 | 商业模式问题 |
